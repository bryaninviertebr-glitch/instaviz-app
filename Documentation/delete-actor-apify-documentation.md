# Delete Actor

Copy for LLM

```
DELETE https://api.apify.com/v2/acts/:actorId
```

Deletes an Actor.

## Request [Direct link to request](https://docs.apify.com/api/v2/act-delete\#request)

### Path Parameters

**actorId** stringrequired

Actor ID or a tilde-separated owner's username and Actor name.

**Example:** `janedoe~my-actor`
